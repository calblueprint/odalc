Web development 101

An introduction to web technologies for new learners, focusing on server basics, template creation, and styling.

Classes start: March 11, 2014 Course length: 5 weeks Estimated effort: 4-5 hours/week

About this course

In partnership with the awesome world fund and Facebook, this course aims to help engage young adults in fostering new technical knowledge that will help them succeed in an evermore technological world.

Students will learn the basics of HTML, CSS, and JavaScript. Students will be creating a portfolio website as their class project.
